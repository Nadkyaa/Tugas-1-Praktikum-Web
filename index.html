What We Do

Interface Design

Desain antarmuka pengguna (UI Design) adalah proses merancang tampilan visual dan interaksi pengguna dengan perangkat lunak, aplikasi, atau perangkat keras, dengan fokus pada estetika, kemudahan penggunaan, dan pengalaman pengguna yang menyenangkan

Web Application

Web application adalah perangkat lunak yang dapat diakses melalui browser web. Aplikasi ini berjalan di server jarak jauh dan tidak perlu diunduh ke perangkat pengguna.

Software Development

Software development atau pengembangan perangkat lunak adalah proses untuk membuat perangkat lunak, seperti aplikasi, situs web, atau sistem operasi. Proses ini melibatkan perencanaan, desain, pemrograman, pengujian, dan pemeliharaan
